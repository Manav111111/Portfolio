# Manav Gupta — Developer Portfolio 🚀

A premium, dark-themed developer portfolio built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**. Showcasing my projects, skills, work experience, and achievements as a Full Stack AI Developer.

## 🔗 Live Preview

> Coming soon on Vercel

## ✨ Features

- **Premium Dark UI** — Soft dark background (`#0B0F19`) with a curated purple (`#8B5CF6`) + indigo (`#6366F1`) accent palette
- **3D Hero Image Flip** — Auto-flipping hero image with glow emission effects
- **Animated Typing Effect** — Cycles through _Full Stack Developer_, _AI Builder_, _Problem Solver_
- **Interactive Skills Orbit** — SVG-based animated skills wheel organized by category (Frontend, Backend, AI/ML, Tools)
- **Project Showcase** — Featured projects with tech badges, highlights, Live Demo & Source Code buttons
- **Timeline Experience** — Career journey displayed as an interactive timeline with glowing dots
- **Achievements Section** — Hackathon wins and awards with trophy/medal/lightning icons
- **Stats Dashboard** — 15+ Projects, 10+ Clients, 2+ Years Experience, 5+ AI Models
- **Contact CTA** — Email, WhatsApp, and LinkedIn contact buttons
- **Resume Download** — Navbar resume button for quick access
- **Fully Responsive** — Works seamlessly on all devices
- **SEO Optimized** — Meta tags, Open Graph, structured data
- **Vercel Analytics** — Performance tracking built-in

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Manav111111/Portfolio.git

# Navigate to the project
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## 🎨 Design System

| Token | Value | Usage |
|-------|-------|-------|
| Background | `#0B0F19` | Soft dark, not pure black |
| Text | `#E5E7EB` | High contrast white |
| Muted Text | `#9CA3AF` | Secondary text |
| Primary | `#8B5CF6` | Buttons, accents, highlights |
| Secondary | `#6366F1` | Gradients, tags |
| Green | `#22C55E` | Status dot, checkmarks |
